# 第三章
10. 反射向量（应该不会考吧）
N为法向量，P为入射向量的-1倍
Q = 2N（N・P）-P
11. 直线方程
- 一般式： Ax + By + C = 0
- 点斜式： y = kx + b
- 截距式： y = y' + k(x - x')
- (y - y') / (y'' - y') = (x - x') / (x'' - x')
- 参数方程：
  x = x' + t(x'' - x') 
  y = y' + t(y'' - y') 
  z = z' + t(z'' - z') 
12. 平面方程和法向量
法向量N(A, B, C) = (p - p') × (p'' - p)
- 一般式：Ax + By +Cz +D = 0
- (A, B, C) ・ (x - x', y - y', z - z') = 0
**判断点在面的正面还是反面：**
将点的坐标代入平面方程，大于0为正面，小于0为反面。
**判断直线与平面的交点：**
用参数方程代入，求t
**求点到平面距离**：
找平面上任意一点，作与目标点的向量，再点乘单位法向量
